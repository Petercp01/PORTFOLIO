NAVES PARA REDEROS EN LAXE

Este proyecto se ha realizado durante el segundo cuatrimestre del curso 2021-22. Se proponía realizar una nave para almacenar los aparejos de pesca, en la que se contasen con 20 departamentos cerrados pero con ventilación natural. 

El proyecto se plantea como una serie volúmenes con forma de arco de diferentes tamaños que se alternan para formar un recorrido interior cubierto. Cada uno de estos cuerpos se ejecuta mediante un sistema prefabricado estructural de chapas de hierro corrugado galvanizado, de este destaca su rápido montaje y desmontaje así como su resistencia al contar con elementos plegados. En caso de necesitar su desmontaje sólo queda en el lugar la cimentación. 

Dentro de estos volúmenes abiertos se sitúan los almacenes para los rederos, asociados a estos un espacio exterior cubierto permite el refugio de los trabajadores los días de lluvia o mal tiempo para reparar los aparejos. 

En uno de los volúmenes se plantea una cafetería o sala de reuniones para los usuarios del puerto siendo este el único cuerpo totalmente cerrado del conjunto.
